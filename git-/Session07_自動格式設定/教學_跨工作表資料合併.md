# 教學：跨工作表資料合併與 For 迴圈初探

在這堂課中，我們要學習一個非常實用的技巧：**如何從一個試算表中，把好幾個不同工作表（Sheet）的資料抓出來，然後全部合併在一起！**

這段程式碼看起來有點長，而且包含了一個大家可能覺得陌生的東西——`for` 迴圈。別擔心，我們會把它拆成好幾個小步驟，一步一步帶大家理解。

## 我們的目標程式碼
我們要學習的函式長這樣：

```javascript
function getDataFromSpreadsheet(ssID){
  var ss = SpreadsheetApp.openById(ssID)
  var allsheets = ss.getSheets();
  var data = []
  
  for(var s in allsheets){
    var sheet = allsheets[s];
    if(
       (sheet.getName() == "Summary") || 
       (sheet.getName() == "Data") || 
       (sheet.getName() == "Sheet1") ||
       (sheet.getName() == "Sheet2")
    ){
      var lastrownum = sheet.getLastRow();
      var values = sheet.getRange("A2:L" + lastrownum).getValues();
      data = data.concat(values)      
    }
  }
  return data;
}
```

---

## 學習步驟拆解

### 步驟一：打開試算表，取得所有工作表
一開始，我們需要先告訴程式要處理哪一個試算表檔案。

```javascript
  var ss = SpreadsheetApp.openById(ssID); // 透過 ID 打開指定的試算表檔案
  var allsheets = ss.getSheets();         // 取得這個檔案裡面「所有的工作表」
```
> **💡 想像一下**：`allsheets` 就像是一個抽屜櫃，裡面裝了這份試算表的所有分頁（例如：Sheet1, Sheet2, Data... 等等）。

### 步驟二：準備一個大箱子裝資料
因為我們要收集很多個工作表的資料，所以我們需要先準備一個「空箱子」。

```javascript
  var data = []; // 建立一個空的陣列 (Array)，用來裝最後合併好的所有資料
```

### 步驟三：認識 `for...in` 迴圈 (最重要的一步！)
現在，我們有一個裝滿工作表的櫃子 `allsheets`。我們需要**一個一個**把它們拿出來檢查。這時候就需要請出 `for` 迴圈了！

```javascript
  for(var s in allsheets){
    var sheet = allsheets[s]; // 這是迴圈內部
    // ... 我們會對每一個 sheet 做一些事情
  }
```
**什麼是 `for` 迴圈？**
* 想像你面前有一排抽屜（這就是 `allsheets`）。
* `for(var s in allsheets)` 的意思就是：「**請幫我從第一個抽屜開始，一個一個打開，直到最後一個抽屜為止。**」
* 變數 `s` 就是目前的「**抽屜編號**」（例如：第 0 個、第 1 個、第 2 個...）。
* `var sheet = allsheets[s];` 的意思就是：「**請把第 `s` 個抽屜裡面的工作表拿出來，並把它命名為 `sheet`。**」

有了這個迴圈，不管試算表裡有 5 個工作表還是 50 個工作表，程式都會自動幫我們一個一個檢查，不用我們手動寫 50 次！

### 步驟四：過濾我們想要的工作表
雖然迴圈會幫我們把每個工作表拿出來，但我們不一定每個都要。我們只想要特定名稱的工作表。

```javascript
    if(
       (sheet.getName() == "Summary") || 
       (sheet.getName() == "Data") || 
       (sheet.getName() == "Sheet1") ||
       (sheet.getName() == "Sheet2")
    ){
      // 如果工作表名稱是上面四個的其中一個，才會執行這裡面的動作
    }
```
* `sheet.getName()`：取得目前這個工作表的名稱。
* `||`：這個符號在程式裡代表「**或者 (OR)**」。
* 整段的意思是：「如果這個工作表的名字叫做 "Summary" **或者** "Data" **或者** "Sheet1" **或者** "Sheet2"，我們才要處理它。」如果是別的名字（例如 "暫存區"），程式就會直接跳過，繼續檢查下一個抽屜。

### 步驟五：抓取資料
確定是我們想要的工作表後，接下來就要把裡面的資料挖出來了。

```javascript
      var lastrownum = sheet.getLastRow(); // 找到這個工作表有資料的最後一列是第幾列 (例如: 第 50 列)
      var values = sheet.getRange("A2:L" + lastrownum).getValues(); // 抓取 A2 到 L50 這個範圍內的所有資料
```
* **為什麼是 `A2`？** 通常第一列 (Row 1) 是標題列，我們抓資料時通常只想要內容，所以從第二列開始抓。
* **為什麼是 `+ lastrownum`？** 這樣寫程式才會自動根據每個工作表資料的多寡，動態決定要抓到哪一列。假設有 50 列，就會變成抓取 `A2:L50`，這樣就不會抓到一堆空白行了。

### 步驟六：把資料倒進大箱子裡
我們已經把當前工作表的資料抓出來放進 `values` 裡了，現在要把它加到步驟二準備好的空箱子 `data` 裡面。

```javascript
      data = data.concat(values); // 將 values 的資料接在 data 的後面
```
* `concat` 是一個可以把兩個陣列（裝資料的箱子）**連接在一起**的魔法指令。
* 每當迴圈跑過一個符合條件的工作表，就會把新抓到的資料 `values` 接著裝進原本的 `data` 大箱子裡。

### 步驟七：大功告成，交出成果
當 `for` 迴圈把所有的工作表都檢查完，該抓的資料都抓完之後，迴圈就會結束。

```javascript
  return data; // 把裝滿所有資料的大箱子交出去給呼叫這個函式的人
```

---

## 總結複習
這段程式的流程就像是：
1. 給我一個試算表檔案。
2. 我準備一個大箱子 `data`。
3. 我**用 `for` 迴圈一個一個檢查**裡面的每一頁工作表。
4. **如果**名字是我要的，我就把那一頁從 A2 開始到底的資料抓出來。
5. 把抓出來的資料全部倒進大箱子 `data` 裡。
6. 檢查完所有工作表後，把裝滿的大箱子交給你。

希望透過這樣的拆解，大家能對 `for` 迴圈以及這段如何跨工作表合併資料的程式碼有更清楚的了解！

## 🚀 進階教學：陣列與巢狀迴圈 (Nested For Loop)

當我們處理 Google 試算表的資料時，常常會遇到「一維陣列」和「二維陣列」，有時候我們甚至需要用「迴圈包著迴圈」（巢狀迴圈）來處理它們。讓我們一起來搞懂這兩個進階觀念！

### 觀念一：一維陣列 vs 二維陣列

#### 1. 一維陣列 (1D Array)
一維陣列就像是一條**單純的清單**或是一排排隊的人。裡面裝著一個個獨立的資料。
```javascript
// 這是一個一維陣列
var fruits = ["蘋果", "香蕉", "橘子", "芭樂"];
```
如果你要拿「橘子」，程式碼是 `fruits[2]` (記住，陣列的編號是從 0 開始數的！)。

#### 2. 二維陣列 (2D Array)
二維陣列就像是一個**表格**或是一個**置物櫃**。它是「陣列裡面還包著陣列」。
**⭐ 重要觀念：在 Google Apps Script 裡，只要你用 `getValues()` 抓出來的範圍資料，全部都會是二維陣列！**

```javascript
// 這是一個二維陣列 (想像它是一個 3 列 x 2 欄 的表格)
var tableData = [
  ["王小明", 90], // 第 0 列 (Row 0)
  ["陳小華", 85], // 第 1 列 (Row 1)
  ["林阿珠", 95]  // 第 2 列 (Row 2)
];
```
* **外層的陣列**代表「列 (Row)」。
* **內層的陣列**代表每一列裡面的「欄 (Column)」。
如果你想拿到陳小華的分數 (85分)，你需要兩個中括號：`tableData[1][1]` (先找第 1 列，再找裡面的第 1 欄)。

### 觀念二：巢狀 For 迴圈 (迴圈裡的迴圈)

如果我們有一個二維陣列（表格資料），我們想要「檢查每一個儲存格」的內容，單一一個 `for` 迴圈是不夠的。這時候我們就需要**巢狀迴圈**！

**想像一個情境**：你是一棟大樓的管理員，你要檢查每一層樓的每一個房間有沒有關燈。
* **外層迴圈**：負責一層一層樓往上爬 (處理列 Row)
* **內層迴圈**：負責每一層樓裡，一間一間房間檢查 (處理欄 Column)

#### 📝 進階程式碼範例：找出表格中不及格的分數
假設我們抓出了一份成績單的二維陣列資料，我們想要用巢狀迴圈把不及格的分數印出來。

```javascript
function checkScores() {
  // 假設這是從試算表抓出來的資料 (二維陣列)
  var scoresData = [
    ["姓名", "國文", "數學", "英文"], // 標題列 (第 0 列)
    ["王小明", 90, 55, 80],         // 資料列 (第 1 列)
    ["陳小華", 40, 85, 95],         // 資料列 (第 2 列)
    ["林阿珠", 60, 70, 59]          // 資料列 (第 3 列)
  ];

  // 外層迴圈：負責一列一列往下看 (r 代表 row)
  // 注意：我們讓 r 從 1 開始，是為了跳過標題列 (第 0 列)
  for(var r = 1; r < scoresData.length; r++) {
    
    var row = scoresData[r]; // 拿出目前這一列的資料，例如 ["王小明", 90, 55, 80]
    var name = row[0];       // 每一列的第一個資料是姓名
    
    // 內層迴圈：負責檢查這一列裡面的每一個分數 (c 代表 column)
    // 注意：我們讓 c 從 1 開始，因為第 0 欄是姓名，不是分數
    for(var c = 1; c < row.length; c++) {
      
      var score = row[c]; // 拿出這一個儲存格的分數
      
      // 檢查分數是否不及格 (< 60)
      if(score < 60) {
        // 如果不及格，就印出訊息
        Logger.log(name + " 有一科不及格，分數是：" + score);
      }
    }
  }
}
```

**這個巢狀迴圈的執行順序是：**
1. **外層迴圈**走到第 1 列 (`r=1`，王小明那一列)。
2. **內層迴圈**開始啟動：檢查王小明的國文(`c=1`)、再檢查數學(`c=2`)、再檢查英文(`c=3`)。數學不及格，印出訊息。
3. 內層迴圈結束，**外層迴圈**走到第 2 列 (`r=2`，陳小華那一列)。
4. **內層迴圈**再次啟動：檢查陳小華的國文(`c=1`)、數學(`c=2`)、英文(`c=3`)。國文不及格，印出訊息。
5. 依此類推，直到外層迴圈把所有的列都檢查完畢。
