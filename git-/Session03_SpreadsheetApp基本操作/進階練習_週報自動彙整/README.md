# 進階練習：🤖 多部門週報自動彙整系統

> 對應 Session 3 ｜ 辦公室自動化 AI 主題

## 🎯 練習目標

- 自動為 5 個部門建立標準化週報模板工作表
- 週五自動彙整所有部門週報到「總覽」
- 觸發器實現全自動：週一建立 → 週五彙整

## 💼 情境

公司有 5 個部門，每週需要繳交週報。過去用 Email 彙整，常漏收。現在改用一份 Google Sheet，每個部門一個工作表，週五自動彙整。

## 📖 操作步驟

1. 貼入 `code.gs`，使用選單「建立本週週報」
2. 各部門在對應工作表填寫（可多人協作）
3. 週五執行「彙整所有週報」或讓觸發器自動執行

## 🧠 進階挑戰

1. 自動 Email 通知未填寫週報的部門
2. 建立週報完成率統計圖表
