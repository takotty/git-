# 進階練習：🤖 智慧會議記錄管理系統

> 對應 Session 1 ｜ 辦公室自動化 AI 主題

---

## 🎯 練習目標

透過建立一套完整的會議管理系統，整合 Session 1 學到的所有技能：
- **變數與物件**：管理會議設定與類型對照
- **函數**：模組化的模板生成邏輯
- **巨集**：一鍵標記出席、完成、分隔線
- **觸發器**：自動建立選單、每週提醒

---

## 💼 情境說明

你是公司的行政助理，每天要處理大量會議記錄。過去都是手動複製模板、調整格式、追蹤決議，費時費力。現在要用 GAS 打造一套 **智慧會議管理系統**：

1. 📋 根據會議類型（週會/專案/客戶/部門/臨時）自動生成對應模板
2. ⚡ 用巨集快速操作（標記出席、標記完成）
3. ⏰ 每週一自動盤點未完成決議，發送提醒
4. 📊 一鍵產生會議統計報表

---

## 📖 操作步驟

### Step 1：建立試算表

1. 建立新試算表 `智慧會議管理系統`
2. 貼入 `code.gs`，儲存
3. 執行「初始化範例資料」建立會議排程
4. 重新整理頁面，使用選單

### Step 2：建立會議記錄

1. 點選「🤖 智慧會議管理」>「📋 建立會議記錄」
2. 依序輸入：會議類型、主題、主持人
3. 系統自動建立對應模板的工作表

### Step 3：使用巨集

1. 點選出席人員列 → 快速操作 >「📝 標記出席」
2. 點選決議列 → 快速操作 >「✅ 標記完成」

### Step 4：設定觸發器

1. 點選「⏰ 設定每週自動提醒」
2. 系統會在每週一盤點未完成決議

## ❓ FAQ

### Q：如何新增自訂會議類型？
修改 `會議設定.會議類型` 物件，加入新的類型、顏色和圖示。再到 `_取得預設議程` 加入對應的預設議程。

### Q：巨集可以綁定快捷鍵嗎？
在 Google Sheet 中，選擇「擴充功能 > 巨集 > 管理巨集」，可以為已存在的巨集設定快捷鍵（Ctrl+Alt+Shift+1~10）。
