## 📖 步驟教學

### Step 1：建立 Google Sheet 並匯入範例資料

1. 開啟 [Google Sheets](https://sheets.google.com)
2. 點選「**+ 空白**」建立新試算表
3. 將試算表命名為 `GAS課程練習_Session01`
4. 將工作表名稱改為 `成績單`（點擊底部工作表標籤，右鍵 > 重新命名）
5. 開啟 `sample_data.csv`，將資料複製貼入 A1 儲存格

> 💡 **提示**：也可以透過「檔案 > 匯入」直接匯入 CSV 檔案

### Step 2：開啟 Apps Script 編輯器

1. 在 Google Sheet 中，點選選單列「**擴充功能**」
2. 選擇「**Apps Script**」
3. 會開啟一個新的分頁，這就是 Apps Script 編輯器
4. 預設會有一個 `程式碼.gs` 檔案，內含空的 `myFunction()`

### Step 3：貼上程式碼

1. 刪除預設的 `myFunction()` 程式碼
2. 將 `code.gs` 的**全部內容**複製貼入編輯器
3. 按下 `Ctrl + S`（或 `Cmd + S`）儲存

### Step 4：執行程式碼

1. 在編輯器上方的函數選單中，選擇要執行的函數
2. 建議先執行 `初始化範例資料`（如果你還沒手動輸入資料）
3. 第一次執行時，Google 會要求授權，請點選「審查權限」>「允許」
4. 執行後，回到 Google Sheet 查看結果

### Step 5：查看執行記錄

1. 在 Apps Script 編輯器中，點選左側的「**執行項目**」圖示
2. 或點選「**檢視 > 執行紀錄**」查看 `Logger.log` 的輸出

### Step 6：體驗自訂選單

1. 回到 Google Sheet，重新整理頁面（F5）
2. 等待載入後，會看到選單列多出「📚 課程工具」
3. 點選選單中的各功能試用

## 💻 程式碼解說

### 1. 變數與資料型別

```javascript
// 三種變數宣告方式
var 姓名 = "王小明";     // 舊式寫法，函數作用域
let 年齡 = 25;           // 新式寫法，區塊作用域，可重新賦值
const 公司 = "ABC公司";   // 常數，不可重新賦值
```

> **建議**：優先使用 `const`，需要重新賦值時用 `let`，避免使用 `var`

### 2. 函數定義與呼叫

```javascript
// 定義函數：接收參數，回傳結果
function 計算加總(a, b) {
  return a + b;
}

// 呼叫函數
let 結果 = 計算加總(10, 20);  // 結果 = 30
```

### 3. 流程控制 (if-else)

```javascript
if (平均 >= 90) {
  等第 = "優等";
} else if (平均 >= 80) {
  等第 = "甲等";
} else {
  等第 = "不及格";
}
```

### 4. 巨集的本質

巨集本質上就是一段 Apps Script 程式碼。透過「錄製巨集」功能：
- Google 會**記錄你的操作**並自動產生程式碼
- 你可以**手動修改**這段程式碼來客製化行為

### 5. 觸發器 (Trigger)

```javascript
// onOpen() 是「簡單觸發器」
// 當試算表開啟時會自動執行
function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu("📚 課程工具")
    .addItem("功能名稱", "函數名稱")
    .addToUi();
}
```

## ❓ 常見問題 FAQ

### Q：執行時出現「授權必要」怎麼辦？
**A**：第一次執行需要授權。點選「審查權限」→ 選擇你的 Google 帳號 → 點選「進階」→「前往（不安全）」→「允許」。

### Q：Logger.log 的輸出在哪裡看？
**A**：在 Apps Script 編輯器中，點選左側的「執行項目」圖示，或選擇「檢視 > 執行紀錄」。

### Q：為什麼自訂選單沒有出現？
**A**：需要重新整理 Google Sheet 頁面。`onOpen` 函數只在開啟（或重新整理）試算表時執行。

### Q：修改程式碼後要怎麼再次執行？
**A**：先儲存（Ctrl+S），然後在函數選單中選擇要執行的函數，點選「▶ 執行」按鈕。

### Q：var、let、const 到底差在哪？
**A**：
| 宣告 | 作用域 | 可重新賦值 | 建議使用時機 |
|------|--------|------------|-------------|
| `var` | 函數 | ✅ | 不建議使用 |
| `let` | 區塊 | ✅ | 需要改變值時 |
| `const` | 區塊 | ❌ | 值不會改變時 |
