## 💻 程式碼解說

### filter 篩選語法

```javascript
// 保留符合條件的元素
var 結果 = 陣列.filter(function(item) {
  return item["金額"] > 5000;  // 條件：回傳 true 保留
});
```

### sort 排序語法

```javascript
// 升冪排序
陣列.sort(function(a, b) { return a - b; });

// 降冪排序
陣列.sort(function(a, b) { return b - a; });

// 文字排序
陣列.sort(function(a, b) {
  if (a < b) return -1;
  if (a > b) return 1;
  return 0;
});
```

### 分組統計模式

```javascript
var 統計 = {};
資料.forEach(function(item) {
  var key = item["業務"];
  if (!統計[key]) 統計[key] = { 筆數: 0, 總額: 0 };
  統計[key].筆數++;
  統計[key].總額 += item["金額"];
});
```

## ❓ 常見問題 FAQ

### Q：`filter` 和 `for` 迴圈有什麼差別？
**A**：功能相同，但 `filter` 語法更簡潔、更具表達力。建議有支援時優先使用 `filter`。

### Q：`sort` 會修改原陣列嗎？
**A**：會！`sort` 是原地排序。如果要保留原陣列，先用 `.slice()` 複製再排序。

### Q：處理大量資料時要注意什麼？
**A**：
- 使用批次讀寫（getValues / setValues）
- 避免在迴圈中呼叫 API
- 注意 6 分鐘執行時間限制
- 大量字串操作會消耗記憶體
