# 進階練習：🤖 智慧採購分析與供應商評比

> 對應 Session 6 ｜ 辦公室自動化 AI 主題

## 🎯 練習目標
- 用 filter + sort 篩選分析 40 筆採購紀錄
- 依供應商分組統計（準時率、退貨率、品質分數）
- 綜合評分公式：品質(40%)+準時率(30%)+退貨率(20%)+價格(10%)
- 自動建立供應商排名報表

## 💼 情境
採購部門每月面對多家供應商，需要客觀的評比機制決定下季合作對象。

## 🧠 進階挑戰
1. 加入價格趨勢分析（同品項不同月份的價格走勢）
2. 自動產生採購建議書（含推薦供應商與預估金額）
3. 結合 Gmail 自動發送評比結果給供應商
