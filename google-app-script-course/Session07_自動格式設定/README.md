## 💻 程式碼解說

### 常用格式方法速查

| 方法 | 說明 | 範例 |
|------|------|------|
| `setFontFamily()` | 字體 | `"微軟正黑體"` |
| `setFontSize()` | 字體大小 | `12` |
| `setFontWeight()` | 粗體 | `"bold"` / `"normal"` |
| `setFontStyle()` | 斜體 | `"italic"` / `"normal"` |
| `setFontColor()` | 字體顏色 | `"#1a73e8"` |
| `setBackground()` | 背景色 | `"#e8f5e9"` |
| `setHorizontalAlignment()` | 水平對齊 | `"left"` / `"center"` / `"right"` |
| `setVerticalAlignment()` | 垂直對齊 | `"top"` / `"middle"` / `"bottom"` |
| `setNumberFormat()` | 數字格式 | `"#,##0"` |
| `setWrap()` | 自動換行 | `true` / `false` |
| `setBorder()` | 框線 | 見下方說明 |
| `merge()` | 合併儲存格 | — |

### 常用數字格式碼

| 格式碼 | 效果 | 範例 |
|--------|------|------|
| `#,##0` | 千分位 | 1,234,568 |
| `#,##0.00` | 千分位+小數 | 1,234,567.89 |
| `0.00%` | 百分比 | 85.67% |
| `NT$#,##0` | 新台幣 | NT$1,234,568 |
| `yyyy/mm/dd` | 日期 | 2026/04/23 |
| `hh:mm:ss` | 時間 | 14:30:00 |

## ❓ 常見問題 FAQ

### Q：`setNumberFormat` 的格式碼從哪裡查？
**A**：與 Google Sheets 的自訂數字格式相同，可參考 [Google 文件](https://developers.google.com/sheets/api/guides/formats)。

### Q：如何同時設定多個格式？
**A**：可以鏈式呼叫：`range.setFontWeight("bold").setFontSize(12).setBackground("#fff")`
