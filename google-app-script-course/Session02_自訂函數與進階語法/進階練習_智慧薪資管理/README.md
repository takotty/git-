# 進階練習：🤖 智慧薪資計算與加班費自動化

> 對應 Session 2 ｜ 辦公室自動化 AI 主題

## 🎯 練習目標
- 建立符合勞基法的加班費自訂函數 `=計算加班費(月薪, 時數, "平日")`
- 建立勞健保、所得稅預扣的自訂函數
- 一鍵批次計算全公司薪資

## 💼 情境
公司有 10 名員工，每月需計算：加班費（平日/假日）、勞健保、所得稅預扣、實發薪資。過去 HR 用 Excel 手算，現在改用 GAS 自動化。

## 📖 操作步驟
1. 建立試算表，貼入 `code.gs`
2. 執行「初始化薪資資料」
3. 可在儲存格直接使用：`=計算加班費(42000, 12, "平日")`
4. 執行「批次計算薪資」一次算完全部

## 🧠 進階挑戰
1. 加入特休假折算薪資的邏輯
2. 自動產生每人的薪資條 PDF
3. 加入年終獎金計算（依績效等級）
