## 💻 程式碼解說

### 1. 自訂函數的關鍵點

```javascript
/**
 * @param {number} 原價 - 參數說明
 * @returns {number} 回傳值說明
 * @customfunction  ← 這個標籤讓函數可以在儲存格使用！
 */
function 計算折扣(原價, 折數) {
  return Math.round(原價 * 折數);
}
```

> ⚠️ **重要**：加上 `@customfunction` 標籤，Google Sheet 才會把它視為自訂函數

### 2. 預設參數值

```javascript
function 計算含稅(未稅金額, 稅率) {
  // 如果沒傳入稅率，使用預設 5%
  if (稅率 === undefined || 稅率 === null || 稅率 === "") {
    稅率 = 0.05;
  }
  return Math.round(未稅金額 * (1 + 稅率));
}
```

### 3. for 迴圈批次處理

```javascript
  // ... 處理邏輯

```

## ❓ 常見問題 FAQ

### Q：自訂函數在儲存格顯示「Loading...」很久？
**A**：自訂函數有 30 秒的執行限制。如果計算太複雜，可能會超時。嘗試簡化計算邏輯。

### Q：為什麼自訂函數不能使用 `SpreadsheetApp`？
**A**：自訂函數的權限受限，不能存取其他工作表或修改儲存格。只能接收參數並回傳結果。

### Q：如何讓自訂函數出現在函數建議清單中？
**A**：確保函數上方有 JSDoc 註解（`/** ... */`）並包含 `@customfunction` 標籤。

### Q：for 迴圈中的 `i = 1` 是什麼意思？
**A**：因為 `資料[0]` 是標題列，我們從 `i = 1` 開始，代表從第一筆資料開始處理。
