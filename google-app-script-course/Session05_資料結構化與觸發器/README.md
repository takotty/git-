## 💻 程式碼解說

### 物件回傳值

```javascript
function 取得統計() {
  // ... 計算 ...
  return {           // 用物件回傳多個值
    總人數: 10,
    平均薪資: 50000,
    最高薪資: 68000
  };
}

var 結果 = 取得統計();
Logger.log(結果.平均薪資); // 50000
```

### 二維陣列 → 物件陣列

```javascript
// 原始資料（二維陣列）
// [["姓名","部門"], ["王小明","開發部"], ...]

// 轉換為物件陣列
// [{ 姓名: "王小明", 部門: "開發部" }, ...]

for (var i = 1; i < 資料.length; i++) {
  var obj = {};
  for (var j = 0; j < 標題.length; j++) {
    obj[標題[j]] = 資料[i][j];  // 用標題當 key
  }
  結果.push(obj);
}
```

## ❓ 常見問題 FAQ

### Q：物件和陣列有什麼差別？
**A**：陣列用數字索引 `arr[0]`，適合有序列資料；物件用名稱 `obj.name`，適合具名屬性。通常混合使用：物件陣列 `[{name:"A"}, {name:"B"}]`。

### Q：為什麼要做資料結構化？
**A**：`data[i][3]` 無法一眼看出是什麼欄位，但 `employee["月薪"]` 立刻就懂。程式碼更易讀、更好維護。
